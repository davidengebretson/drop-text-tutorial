  • The index.html file is the file you are currently viewing.
  • The index.css file is the general css styles. This file imports the droptext.css file for droptext specific styles.
  • The droptext.css file holds all the drop-text css styles. This file isimported in by index.css.
  • The droptext.js file is the javascript that gives lfe to the html for dropText.
  • The design.html is my staging area. Here I work out the html without a large overhead. When completed I copy it into the appropiate place in this index.html page.
